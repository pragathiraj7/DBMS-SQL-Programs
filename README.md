# 🗄️ DBMS SQL Programs

This repository contains my DBMS laboratory SQL programs and practice queries written using **Oracle SQL**. It covers basic SQL commands, built-in functions, and query operations that helped me understand fundamental database concepts.

## 📚 Topics Covered

- SELECT Statements
- DUAL Table
- Arithmetic Operations
- Date Functions (`SYSDATE`)
- String Functions
- Mathematical Functions
- Character Functions
- SQL Expressions

## 🛠️ Database Used

- Oracle SQL

## 📁 Files

- `Basic_SQL_Queries.sql` – Collection of basic Oracle SQL queries and functions.

## 🚀 How to Use

1. Open Oracle SQL Developer or Oracle Live SQL.
2. Copy the required SQL query from the `.sql` file.
3. Execute the query and view the output.

## 🎯 Purpose

This repository is created to:
- Practice SQL programming
- Learn DBMS concepts
- Maintain lab programs
- Showcase SQL skills on GitHub
